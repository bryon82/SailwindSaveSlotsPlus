# Changelog

All notable changes to this project will be documented in this file.

- Save slot file management (copy, paste, and delete) and the ability to name/rename your save slot.
- Name save slot when starting new game.

### Updated
- previous and next arrows now gray out when there are no more save slot pages in that direction.

## [v1.0.0] - 2026-06-09

### Added
- A configurable number of save slots.
